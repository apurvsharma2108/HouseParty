
# HouseParty

This Project is created using Django and react where users can have music sessions by creating rooms and hosting rooms.

## Features

- Enjoy Group Sessions on spotify music
- Create Rooms
- Host Rooms
- Easy To use

  
## Appendix

Created Using :
1) Django
2) React Js
3) with the help of spotify Api
